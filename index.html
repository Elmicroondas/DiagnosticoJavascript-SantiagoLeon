¿Qué es una Base de Datos Relacional?

Una base de datos relacional es un sistema que almacena datos organizados en tablas que pueden relacionarse entre sí mediante claves. Son de suma importancia en el mundo debido que toda gran empresa trabaja con ellas.

Estas se pueden bazar en diferentes plataformas, pero siempre mantienen el lenguaje (SQL), algunas de esas plataformas son:

  • MySQL
  • PostgreSQL
  • Oracle Database

Comandos Básicos de SQL

SELECT

Se usa para consultar datos en una tabla.

SELECT * FROM usuarios;

INSERT

Se usa para crear nuevos registros en una tabla.

INSERT INTO usuarios (nombre, edad) VALUES ('Ana', 25);

UPDATE

Actualiza los datos en una tabla.

UPDATE usuarios SET edad = 26 WHERE nombre = 'Ana';

DELETE

Elimina registros de una tabla.

DELETE FROM usuarios WHERE nombre = 'Ana';

Comparación de Comandos

Comando Acción Ejemplo
SELECT Consulta datos SELECT * FROM productos;
INSERT Agrega datos INSERT INTO productos VALUES (...);
UPDATE Modifica datos UPDATE productos SET ...;
DELETE Elimina datos DELETE FROM productos WHERE ...;
